## External Review

Pi review reported one potential release blocker: release notes had moved to
`v0.3.6` while the helper did not expose a version command or release-build
version ldflags. The fix added `codex-orchestrator version`, `--version`, and
`-v`, plus ldflags in release/source install scripts. A release-asset smoke
confirmed `v0.3.6`.

No other blocking findings were reported.

## Evidence Boundary

Release verification is `local` plus GitHub metadata/download `proxy` evidence.
It is not direct proof of Codex App runtime orchestration, daemon behavior,
production deployment, provider state, payment/device/hardware behavior, or
external project correctness.
